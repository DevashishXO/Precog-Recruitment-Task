# models/

Saved model weights from Task 2. 

Only Tier C (DistilBERT + LoRA) is saved here. Tier A (XGBoost) and Tier B (GloVe FFNN) are re-trained from scratch when the notebooks run.

The `distilbert_lora_final/` directory is the model loaded in Tasks 3 and 4.

```
models/
│
├── distilbert_lora_final/            # Final Tier C model — used in Tasks 3 and 4
│   ├── adapter_config.json           # LoRA adapter configuration (r=8, alpha=16, target: q_lin, v_lin)
│   ├── adapter_model.safetensors     # Trained LoRA adapter weights (~1.09% of total parameters)
│   ├── tokenizer.json                # DistilBERT WordPiece tokenizer vocabulary
│   ├── tokenizer_config.json         # Tokenizer settings (max_length=512, etc.)
│   └── README.md                     # Auto-generated by HuggingFace save_pretrained()
│
└── distilbert_lora/                  # Per-epoch training checkpoints (saved during Task 2 training)
    ├── checkpoint-66/                # End of epoch 1
    ├── checkpoint-132/               # End of epoch 2
    ├── checkpoint-198/               # End of epoch 3
    └── checkpoint-264/               # End of epoch 4 (best config. early stopping triggered here)
```

### Loading the model

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification
from peft import PeftModel

tokenizer  = AutoTokenizer.from_pretrained('models/distilbert_lora_final')
base_model = AutoModelForSequenceClassification.from_pretrained(
    'distilbert-base-uncased', num_labels=3
)
model = PeftModel.from_pretrained(base_model, 'models/distilbert_lora_final')
model.eval()
```
