# data/

All data generated and consumed across the five tasks. The pipeline flows left to right:
raw text → cleaned text → class JSONL files → train/val/test splits → feature CSVs.


> **Note:** `data/glove/` is not committed to the repository due to file size. Download instructions are in the root README.

```
data/
│
├── raw/                              # Original Project Gutenberg (PG) text files
│   ├── Frankenstein.txt              # Mary Shelley — Frankenstein (PG #84)
│   ├── Great Expectations.txt        # Charles Dickens — Great Expectations (PG #1400)
│   └── Mathilda.txt                  # Mary Shelley — Mathilda (PG #15238)
│                                     #   Added when Frankenstein alone fell short of 500 segments.
│
├── processed/                        # Cleaned novel texts (Task 0 cleaning pipeline)
│   ├── frankenstein_cleaned.txt
│   ├── great_expectations_cleaned.txt
│   └── mathilda_cleaned.txt
│
├── class1/                           # Class 1 — Human (Victorian novel paragraphs, label=0)
│   ├── class1_shelley.jsonl          # 500 paragraphs pooled from Frankenstein + Mathilda
│   ├── class1_dickens.jsonl          # 500 paragraphs from Great Expectations
│   └── extras/
│       ├── class1_frankenstein.jsonl # Intermediate: Frankenstein-only segments (396)
│       └── class1_mathilda.jsonl     # Intermediate: Mathilda-only segments (160)
│
├── class2/
│   └── class2_generic_ai.jsonl       # Class 2 — Generic AI (Gemini, neutral style, label=1)
│                                     #   500 paragraphs across 10 topics (50 per topic)
│
├── class3/
│   └── class3_mimic_ai.jsonl         # Class 3 — Mimic AI (Gemini, Shelley-style, label=2)
│                                     #   500 paragraphs across 10 topics (50 per topic)
│                                     #   ~45% generated by gemini-2.5-flash (see Task 0 notes)
│
├── splits/                           # Stratified 70/15/15 train/val/test split
│   ├── train.jsonl                   # 1050 records (350 per class)
│   ├── val.jsonl                     # 225 records (75 per class)
│   └── test.jsonl                    # 225 records (75 per class)
│
├── features/                         # Pre-computed stylometric feature matrices (Task 1/2)
│   ├── train_features.csv            # 15 features for all training records
│   ├── val_features.csv              # 15 features for all val records
│   └── test_features.csv             # 15 features for all test records
│
├── glove/                            # GloVe embeddings — NOT IN REPO (download separately)
│   └── glove.6B.300d.txt             # 300-dimensional GloVe vectors (Stanford NLP, 6B tokens)
│
├── checkpoints/                      # Per-topic API generation checkpoints (Task 0)
│   ├── class2_topic00_*.json         # Saved progress for Class 2 generation, topic by topic
│   └── class3_topic00_*.json         # Saved progress for Class 3 generation, topic by topic
│                                     #   These are used to resume after 503 server errors mid-run.
│
├── task4/                            # Genetic Algorithm outputs (Task 4)
│   ├── ga_initial_population.jsonl   # 10 initial paragraphs scored by P(Human)
│   ├── ga_best_per_generation.jsonl  # Best paragraph and scores for each of the 7 generations
│   ├── ga_final_population.jsonl     # All 10 individuals from the final generation
│   └── ga_results.json              # Full history + winning text + final class probabilities
│
├── full_dataset.jsonl                # All 1500 records combined (before splitting)
└── topics.json                       # The 10 shared themes used for AI paragraph generation
```
